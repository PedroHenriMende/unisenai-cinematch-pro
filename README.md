# 🎬 CineMatch Pro

O CineMatch Pro é uma aplicação web desenvolvida com HTML, CSS e JavaScript puro para gerenciamento de um catálogo pessoal de filmes.

O sistema permite cadastrar filmes, visualizar estatísticas do catálogo, pesquisar títulos, filtrar por gênero, assistir trailers diretamente pelo YouTube e remover filmes armazenados localmente no navegador através do LocalStorage.

## ✨ Funcionalidades

* Cadastro de filmes
* Upload de capa por URL
* Inclusão de trailer via YouTube
* Notas pessoais sobre cada filme
* Catálogo responsivo em formato de cards
* Busca por título em tempo real
* Filtro por gênero
* Estatísticas automáticas:

  * Total de filmes
  * Horas assistidas
  * Minutos acumulados
  * Quantidade de gêneros cadastrados
* Reprodução de trailers em modal
* Exclusão de filmes do catálogo
* Persistência de dados com LocalStorage
* Interface moderna inspirada em plataformas de streaming

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* LocalStorage API

## 📚 Conceitos Aplicados

* Programação Orientada a Objetos (POO)
* Manipulação do DOM
* Eventos JavaScript
* Arrays, Filter, Map e Reduce
* Responsividade com CSS Grid e Flexbox
* Persistência de dados no navegador

## 🚀 Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/cinematch-pro.git
```

2. Abra o arquivo `index.html` em qualquer navegador moderno.

Nenhuma instalação adicional é necessária.
